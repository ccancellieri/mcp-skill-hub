About

    

    

      

        The memory-export plugin lets you create portable snapshots of your
        mcp-skill-hub state and restore them on any other instance.
      

      

        

          Snapshot contents
          
            Tasks, teachings, skills, memory segments, hub configuration, and
            enabledPlugins settings. Excludes ephemeral runtime tables (logs,
            cache, session state).
          
        

        

          Privacy
          
            private/ directories are always excluded. Files containing
            known PII tokens are flagged before export. Never share snapshots
            containing sensitive credentials.
          
        

        

          FTS indexes
          
            Full-text search virtual tables (tasks_fts, teachings_fts) are excluded
            from snapshots and automatically rebuilt from base tables on restore.
          
        

        

          Format
          
            .tar.gz with a manifest.json, JSONL table dumps, and .md project memory
            files. Schema version is checked on import.
